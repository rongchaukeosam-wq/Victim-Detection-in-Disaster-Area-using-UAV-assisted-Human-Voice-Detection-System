#  UAV-Assisted Victim Voice Detection System
## Abstract
After natural disasters such as landslides, earthquakes, and building collapses, survivors may be hidden under debris or in low-visibility areas, making conventional visual sensors insufficient. This project develops a UAV-assisted acoustic sensing system for real-time human voice detection in high-noise environments. The system uses an ESP32-S3 microcontroller with a MEMS microphone array for audio acquisition. To reduce UAV rotor noise, the audio signal is processed using a Band-Pass Filter and Wiener filtering. The enhanced signal is then classified using a fine-tuned VGGish CNN model. The system achieved 84.88% validation accuracy, 99.2% human voice recall, and 0.9488 AUC, showing its potential for search-and-rescue applications.
